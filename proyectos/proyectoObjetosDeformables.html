Position Based Dynamics (2020/21)

Objectos 2D y 3D deformables mediante Position Based Dynamics a partir del paper Position Based Dynamics 2007 de Matthias Müller, Bruno Heidelberger, Marcus Hennix y John Ratcliff.
Las restricciones para la estructura se han implementado mediante restricciones de distancia y para las restricciones shear y bend restricciones de triangle bending.

Muelles (2019/20)

Se ha implementado una simulación gráfica de mallas deformables. Para ello se han creado varios nodos o partículas unidas por muelles. Variando las propiedades de estos muelles (constente elástica y amortiguación) se pueden obtener comportamiendo diferentes para conseguir simulaciones de diferentes materiales.
Con ello, hemos simulado algunos ejemplos como banderas y cortinas añadiendole fuerzas de viento, colchonetas elásticas, planchas de metal...
Además, también se han implementado roturas en la malla para darle un mayor realismo. Para ello, se puede especificar una fuerza o longitud máxima que puede soportar cada uno de los muelles. Cuando un muelle supera su límite de rotura deja de hacer efecto produciendo una rotura con muy buenos resultados.
También se ha probado estructuras de una dimensión, utiles para otro tipo de simulaciones como puede ser pelo o cuerdas.
